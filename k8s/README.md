# Kubernetes Manifests

## Files
- namespace.yml - Creates assignment3 namespace
- mysql-secret.yml - Database credentials
- mysql-pv.yml - Persistent Volume for MySQL
- mysql-pvc.yml - Persistent Volume Claim
- mysql-deployment.yml - MySQL Deployment
- mysql-service.yml - MySQL ClusterIP Service
- flask-configmap.yml - Flask app configuration
- flask-deployment.yml - Flask API Deployment
- flask-service.yml - Flask ClusterIP Service
- nginx-configmap.yml - Nginx configuration
- nginx-deployment.yml - Nginx Deployment
- nginx-service.yml - Nginx NodePort Service
